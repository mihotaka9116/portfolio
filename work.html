Design Concept

「リズムと親しみやすさ」をテーマにロゴを設計。単なる記号ではなく、プロダクト（マグカップ等）に展開した際にも生活に馴染むよう、重心を低めに設定したタイポグラフィを採用しました。

Coding Point

CSS Gridを活用し、多種多様なグッズ展開を一覧で分かりやすく配置。画像の遅延読み込み（loading="lazy"）を意識し、表示速度を損なわないよう配慮しました。
